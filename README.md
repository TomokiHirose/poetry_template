# poetry_template

Project template for Python development with Poetry.
Please edit according to your environment.


## install 

```sh
$ poetry install
```

## update

```sh
$ poetry add xxx
$ poetry update
```

## Use Scripts
### Pytest
```sh
$ poetry run pytest
```

### Forat
```sh
# black and isot work for template dirs
$ poetry run format
```

### Lint
```sh
# flake8 and mypy work for template dirs
$ poetry run lint
```

## CreateAPI Documents
```sh
$ poetry run document
```

## Build(create wheel)
```sh
$ poetry build
```
